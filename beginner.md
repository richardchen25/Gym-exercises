# 🟢 Beginner Program (0–6 months)

**Goal:** Learn the movements, build a base of strength and conditioning.
**Equipment:** Machines and cables — stable, easy to learn, safe to push.
**Guideline:** Pick a weight where the last 2 reps of each set are hard but doable with good form. Rest 60–90 sec between sets. Finish every session with 15 min of easy-to-moderate cardio.

## Option A — 5-Day Upper/Lower Split

| Day | Session |
|---|---|
| Mon | Upper Body A |
| Tue | Lower Body A |
| Wed | Rest |
| Thu | Upper Body B |
| Fri | Lower Body B |
| Sat | Arms & Core |
| Sun | Rest |

### Upper Body A
| Exercise | Sets × Reps |
|---|---|
| Chest Press Machine | 4 × 10–12 |
| Lat Pulldown Machine | 4 × 10–12 |
| Shoulder Press Machine | 3 × 10–12 |
| Seated Row Machine | 3 × 10–12 |
| Cable Tricep Pushdown | 3 × 12–15 |
| Machine/Cable Bicep Curl | 3 × 12–15 |

### Lower Body A
| Exercise | Sets × Reps |
|---|---|
| Leg Press | 3 × 10–12 |
| Leg Extension | 3 × 12–15 |
| Seated Leg Curl | 3 × 12–15 |
| Hip Abduction Machine | 2 × 12–15 |
| Calf Raise Machine | 3 × 15–20 |

### Upper Body B
| Exercise | Sets × Reps |
|---|---|
| Pec Deck / Chest Fly Machine | 3 × 10–12 |
| Assisted Pull-Up (or wide Lat Pulldown) | 4 × 10–12 |
| Machine/Cable Lateral Raise | 3 × 12–15 |
| Chest-Supported Row Machine | 4 × 10–12 |
| Rope Tricep Pushdown | 3 × 12–15 |
| Preacher Curl Machine | 3 × 12–15 |

### Lower Body B
| Exercise | Sets × Reps |
|---|---|
| Hack Squat Machine (or Smith Machine) | 3 × 10–12 |
| Glute Kickback Machine | 3 × 12–15 |
| Hip Adduction Machine | 2 × 12–15 |
| Lying Leg Curl | 3 × 12–15 |
| Seated Calf Raise | 3 × 15–20 |

### Arms & Core
| Exercise | Sets × Reps |
|---|---|
| Standing Cable Curl | 3 × 12–15 |
| Overhead Cable Tricep Extension | 3 × 12–15 |
| Cable Crunch | 3 × 15 |
| Rotary Torso Machine | 2 × 12–15/side |

## Option B — 3-Day Full Body (short on time)

Run this Mon/Wed/Fri (or any non-consecutive days):

| Exercise | Sets × Reps |
|---|---|
| Leg Press | 3 × 10–12 |
| Chest Press Machine | 3 × 10–12 |
| Lat Pulldown Machine | 3 × 10–12 |
| Shoulder Press Machine | 2 × 10–12 |
| Seated Leg Curl | 2 × 12–15 |
| Cable Crunch | 3 × 15 |

## Progression

Once you hit the top of the rep range on all sets, add a small amount of weight (usually one pin/plate) and work back up. Full details in [docs/progression-guide.md](../docs/progression-guide.md).
