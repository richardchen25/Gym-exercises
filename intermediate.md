# 🟡 Intermediate Program (6–24 months)

**Goal:** Build strength and size with free-weight compound lifts.
**Structure:** 4-day Upper/Lower. Rest 2–3 min on compounds, 60–90 sec on accessories.
**Prereq:** Learn barbell technique with light weight first — the empty bar is a valid starting point.

| Day | Session |
|---|---|
| Mon | Upper A (bench focus) |
| Tue | Lower A (squat focus) |
| Wed | Rest |
| Thu | Upper B (overhead focus) |
| Fri | Lower B (deadlift focus) |
| Sat/Sun | Rest (optional light cardio) |

### Upper A
| Exercise | Sets × Reps |
|---|---|
| Barbell Bench Press | 4 × 6–8 |
| Barbell Row | 4 × 6–8 |
| Seated Dumbbell Shoulder Press | 3 × 8–10 |
| Lat Pulldown | 3 × 10–12 |
| Dumbbell Bicep Curl | 3 × 10–12 |
| Cable Tricep Pushdown | 3 × 10–12 |

### Lower A
| Exercise | Sets × Reps |
|---|---|
| Barbell Back Squat | 4 × 6–8 |
| Romanian Deadlift | 3 × 8–10 |
| Leg Press | 3 × 10–12 |
| Seated Leg Curl | 3 × 12–15 |
| Standing Calf Raise | 4 × 12–15 |

### Upper B
| Exercise | Sets × Reps |
|---|---|
| Overhead Press (barbell) | 4 × 6–8 |
| Pull-Up (or weighted) | 4 × 6–10 |
| Incline Dumbbell Press | 3 × 8–10 |
| Seated Cable Row | 3 × 10–12 |
| Lateral Raise | 3 × 12–15 |
| EZ-Bar Curl + Overhead Tricep Extension | 3 × 10–12 each |

### Lower B
| Exercise | Sets × Reps |
|---|---|
| Conventional Deadlift | 3 × 5 |
| Hack Squat (or Front Squat) | 3 × 8–10 |
| Walking Lunge | 3 × 10/leg |
| Leg Extension | 3 × 12–15 |
| Seated Calf Raise | 4 × 15–20 |

## Progression

Double progression on everything except deadlift (add ~5–10 lb whenever all reps are clean). Deload every 5–6 weeks. See [docs/progression-guide.md](../docs/progression-guide.md).
